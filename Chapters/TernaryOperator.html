Ternary Operator
In C++, we have an operator which can be considered as shorthand of if else statement.

The syntax for ternary operator is :

expression1 ? expression2 : expression3

expression1 is the condition, and if it us true expression2 is executed else expression3

For example :

(2>3) ? cout << "True" : cout << "False"

This will print False.

This operator is mainly used for assignment purposes, as :

int x = (2>3) ? 10 : 20

The value stored in x is 20.
