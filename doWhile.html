do while
In do while loops the loop execution is terminated on the basis of test condition. The main difference between do while loop and while loop is in do while loop the condition is tested at the end of loop body.

Note: In do while loop the loop body will execute at least once irrespective of test condition.

Syntax :

initialization expression; do { // statements } while (test_expression);

The statements must have an update statement aslo.

Notice the semi – colon(“;”) in the end of loop.
