Increment and Decrement
There are two more arithmetic operators increment(++) and decrement(--) operators. These require basic understanding of assignment operators. These are used to increment or decrement the value of a variable by one. The syntax to use these operators is : ++x, --x, x++, x-- For example, int x = 10 ; x++; Now the value of x is 11. ++x will also do the same thing, then why there are two different syntaxes for same thing. When it comes to assignment with increment these two work different. If we write, int x = 10 , y, a=10, b ; y=++x; b=a++; If we print the values of y and b we will see that y is 11 but b is 10, although we did almost same thing for them. This is because the position of ++ operator. When placed before the variable, the increment or decrement operator works before the value of variable is used, but when placed after, the value of variable is used first and then incremented or decremented. In case of y=++x the value of x is incremented first and then assigned to y but in case of b=a++ the value of a is assigned first and then incremented. Depending on the position the operator have prefix pre and post.
