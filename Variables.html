Variables
In programming we often word on data. For that we need to store data. And for that you need to reserve some space in computer's memory. Reserving space is quite easy. All you need to do is follow the given steps.

First you need to tell the computer which type of data you want to store. You do it by writing the keyword for that datatype. How much memory should be reserved for this data is determined by the datatype. We discussed the size of datatype in some previous lesson.

Then you need to give a label to that reserved position so that you can easily access to that data just by mentioning the label. This label is nothing but Identifier for that memory location, same thing we learnt in previous lesson.

And then a semicolon. That would be enough to reserve a portion of memory to store data.

Let us see an example :

int x ;

This statement reserves a portion of memory of size 4 bytes to store integer data and labels that portion as x.

This reserved portion is what we call a Variable.

How to assign data to this variable?

Just write =value after the identifier, either in the same line of declaration or in a different line.

For instance :

int x = 10;

OR

int x ;
x = 10 ;

Both assign the value 10 to the variable x.

There is slight difference in these two ways.

In the first way as soon as the variable is created the value 10 is assigned to it whereas in second way, until 10 is assigned to the variable, it holds some garbage value.

Printing value of Variable on Screen

To print the value of a variable on screen you need to write :

cout << identifierOfTheVariable ;

For instance :

cout << x ;

It will print the value of x on screen.
