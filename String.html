String
In fundamental datatypes, we have learnt numbers, characters etc. But what if we want to store a message like India is my Country? There are multiple ways of achieving the same. Some are form C, which are not as easy C++ allows us to do it.

In C++, we have a something which is not a fundamental datatype but works almost the same way like other fundamental datatypes. This thing is called String.

To store a message of multiple characters we use String, just like we use fundamental datatypes. But remember one thing strings are always written in double quotes.

To declare a string, the syntax is :

String identifier ;

For instance :

String myCountry ;

The above line declares a string with identifier myCountry.

To assign value, you need to write :

myCountry = "India" ;
