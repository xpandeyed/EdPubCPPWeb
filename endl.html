endl
In the last lesson you learnt how to use cout to print data on screen. But there is one more thing you will frequently need while printing data on screen. That is changing the lines. endl will help you there. If you think writing cout << "Statement one" ; cout << "Statement two" ; will print the two statements in two different lines. It will not. Because you have instructed computer to print two statements but you have not instructed it to change the line. This is where you need endl. Wherever you want to insert a new line you need to write : << endl For instance, cout << "Hello World" << endl ; will insert a new line after Hello World. Not only this, if you want to print multiple messages in different lines, you do not need to write cout multiple times. You can write : cout << "Statement one" << endl << "Statement two" ; The functionality of cout will remain until a semicolon is encountered. This means you can print any number of messages in multiple lines using only one cout. One more thing, endl is also an object like cout and we need iostream and standard namespace for endl also. But once you mention those files at the top you do not need to mention those files again and again.
