if else if
While writing a program we often need to execute some statements if a certain condition is met. For such cases C++ provides us if statement. The syntax : if keyword is written, followed by the condition which we want to check in round brackets and then the statements we want to execute. if(condition) { //some code here } This some code will only be executed only when the condition is true. If the condition is false the computer will not execute the block. The if statement is not limited to this only, it can have an else block also. This else block is put after the if block and is executed only if the condition is false. if(condition) { //some code } else { //some code } This does not end here we can have multiple conditions also. if(condition1) { } else if(condition2) { } . . . else { } The else statement is optional in each case. One more thing to notice is that, the conditions are checked in same sequence as they appear and once a condition is met all of the next conditions are ignored.
