Logical Operators
Logical operators are used to check whether a given set of expressions are true or not, in which each expression must have a value either true or false. Following are logical operators, their names, their syntax and values given in different cases : && Name : Logical And Syntax : expression1 && expression2 Takes two expressions. Gives true if left and right both the expressions are true else false. For example : (10>20)&&(20>50) gives false (20>10)&&(20<50) gives true (20<10)&&(20<50) gives false || Name : Logical Or Syntax : expression1 || expression2 Takes two expressions. Gives true if any of the expressions is true. For example : (10>20)||(20>50) gives false (20>10)||(20<50) gives true (20>10)||(20>50) gives true ! Name : Logical Not Syntax : !expression Takes only one expression. Comes before the expression. Gives true if the expression is false and vice-versa. For example : !(10<20) gives false !(20<10) gives true
